Lists (@for)

Use @for to repeat a chunk of HTML for each item in a collection. Angular needs a track expression so it can reuse DOM efficiently when rows move, appear, or disappear ( always give each row a stable identity — often an id field). 

{{ loopSnippet }}
    @for (task of tasks; track task.id) {
  • {{ task.id }} {{ task.label }}
    • }

Related idea: @empty blocks after @for handle “no items” UX — see the Angular docs when you need an empty state.
