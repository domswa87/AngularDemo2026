RxJS (Observable + async pipe)

Angular ships with RxJS. An Observable is a lazy stream: nothing runs until something subscribes. In templates, the async pipe subscribes for you and passes the latest value into the view — and it unsubscribes when the component is destroyed (fewer memory leaks).
